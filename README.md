### Prerequisites

you need to have python and git for setup

```Linux
apt install python python-pip git -y
```

### Installing

A step by step installing and setup Password Generator scripted.

```Linux
git clone https://github.com/LetnanGM/passwd_general_generator.git
```

then change directory to 'passwd_general_generator' and execute 'app.py'

```Linux
cd passwd_general_generator
python app.py
```

output example :

```output example
Your password is : android2012
```
